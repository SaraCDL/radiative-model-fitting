# Gamma-Ray Spectral Model Fitting

A modular Python framework for spectral model fitting, statistical model comparison, and scientific visualization.

## Features

- Power Law
- Exponential Cutoff Power Law
- Broken Power Law
- Log-Parabola

## Statistical Tools

- χ²
- Reduced χ²
- AIC
- AICc
- BIC

## Technologies

- Python
- NumPy
- SciPy
- Astropy
- Matplotlib
- Naima

## Example

python -m examples.fit_all_models

## Data

The example dataset included in `data/HWC_J1908_06.txt` contains publicly available spectral measurements published by the HAWC Collaboration.

These data are provided solely for demonstrating the model-fitting and statistical analysis workflow implemented in this repository.
